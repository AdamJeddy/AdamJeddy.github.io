My Projects

Discord Bot

Me and my friends had decided to make a bot on discord using discord.js which is a powerful node.js module that allows you to interact with the Discord API very easily. Discord is a software that lets you create/join servers in which you can talk, chat, hang out and stay close with communities and your friends. You can check out more about our bot here.


Designing a Network

This was a team project for my CSIT127 subject so the aim of this project is to successfully create a network design of a new Wollongong University Campus in Abu Dhabi. As per the requirements, there are two floors, amongst them we used the ground floor for the faculty and admin offices, the top floor for the classrooms and labs. There are 10 classrooms, 4 labs, 8 faculty offices, 5 admin offices and 1 datacenter. This is the overview of the whole network.


Working with AR filters

I really got interested in augmented reality (AR) and wanted to try it out myself so during the summer break I took some time out learning how to model simple objects and how to use Spark AR, a software by Facebook which helps you easily export your filters to all social media platforms. I had come up with an idea of having button as eyes for the filter (I took the inspiration from the animated movie Coraline), so I had quickly made a model which went through 5 changes before the final one, then imported it in the software and wrote some lines of code so that the buttons can stay on the eyes. This is how my filter performed since its release.
